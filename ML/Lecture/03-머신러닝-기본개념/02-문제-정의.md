# 2강. 문제 정의 — 회귀 vs 분류, 특성과 타깃

지도학습을 시작하기 전, **"내가 무엇을 예측하려는가"** 를 명확히 정의해야 합니다.
이 정의가 알고리즘 선택, 손실함수, 평가 지표를 전부 결정합니다.

---

## 1. 특성(feature)과 타깃(target)

머신러닝 데이터는 보통 표(table) 형태입니다.

| 평수 | 방 개수 | 지하철까지 거리(m) | **집값(만원)** |
|:----:|:-------:|:------------------:|:--------------:|
| 24 | 2 | 300 | 35,000 |
| 33 | 3 | 800 | 48,000 |
| 18 | 1 | 150 | 27,000 |

- **특성 (feature, 입력 X)** — 예측에 사용하는 정보. 위 표의 *평수, 방 개수, 거리*.
  - 다른 이름: 독립변수, 입력변수, 속성(attribute)
- **타깃 (target, 출력 y)** — 우리가 맞히려는 값. 위 표의 *집값*.
  - 다른 이름: 종속변수, 정답(label), 레이블

> 📐 수학적으로 머신러닝은 **함수 `f`를 찾는 일**입니다.
> `y ≈ f(X)` — 특성 X를 넣으면 타깃 y가 나오는 함수.
> "학습"이란 데이터를 보고 이 `f`를 점점 정확하게 다듬는 과정입니다.

**핵심 직관:** 데이터의 각 행(row)은 하나의 **샘플(sample)**, 각 열(column)은
하나의 **특성**입니다. 그래서 데이터 전체가 자연스럽게 **행렬(matrix)** 로
표현됩니다. (로드맵 1번 선형대수가 여기서 연결됩니다.)

---

## 2. 회귀 (Regression) vs 분류 (Classification)

지도학습의 갈림길은 **타깃이 무엇이냐**로 결정됩니다.

### 회귀 — 타깃이 "연속적인 숫자"

> **"얼마나?"** 를 예측한다.

- 집값 (35,000만원, 48,000만원, ...)
- 내일 기온 (23.5°C)
- 광고를 봤을 때 예상 매출액

출력이 연속적인 실수이므로 "32,500만원처럼 사이 값"도 자연스럽습니다.

### 분류 — 타깃이 "카테고리(범주)"

> **"어느 것?"** 을 예측한다.

- 스팸 / 정상 (2개 → **이진 분류, binary**)
- 개 / 고양이 / 새 (3개 이상 → **다중 분류, multi-class**)
- 양성 / 악성 종양

출력이 정해진 몇 개의 클래스 중 하나입니다. "1.5번 클래스" 같은 건 없습니다.

### 비교표

| | 회귀 | 분류 |
|---|------|------|
| 타깃 종류 | 연속적 숫자 | 이산적 범주 |
| 예측 질문 | "얼마?" | "어느 것?" |
| 출력 예 | 48,000 | "악성" |
| 대표 알고리즘 | 선형회귀 | 로지스틱 회귀, 결정트리 |
| 대표 평가지표 | RMSE, R² | 정확도, F1, AUC |

---

## 3. 헷갈리기 쉬운 경계

**Q. 별점(1~5점) 예측은 회귀일까 분류일까?**

→ **둘 다 가능합니다.** 1~5의 순서가 의미 있다고 보면 회귀에 가깝고,
"5개의 독립된 등급"으로 보면 분류로 풀 수도 있습니다. 정답은 하나가 아니며,
**문제의 목적과 평가 방식**에 따라 선택합니다.

**Q. 분류 모델이 확률(0.87)을 출력하던데, 그럼 회귀 아닌가?**

→ 아닙니다. 로지스틱 회귀 같은 분류 모델은 내부적으로 **"악성일 확률 0.87"**
같은 연속값을 계산하지만, 최종 출력은 임계값(보통 0.5)을 기준으로
**범주("악성")로 변환**합니다. 목표가 범주이므로 분류입니다.

---

## 4. 문제 정의가 잘못되면 생기는 일

문제 정의는 모든 것의 출발점이라, 여기서 어긋나면 뒤가 전부 무너집니다.

- 분류 문제에 회귀 지표(RMSE)를 쓰면 → 모델 성능을 엉뚱하게 측정
- 타깃을 잘못 고르면 → 비즈니스 목표와 무관한 모델 완성
- 미래에 알 수 없는 정보를 특성에 넣으면 → **데이터 누수** ([4강](04-모델의-일반화.md))

> 💡 **체크리스트**: 새 문제를 만나면 항상 먼저 답하세요.
> ① 지도? 비지도? ② (지도면) 회귀? 분류? ③ 타깃은 정확히 무엇? ④ 특성으로
> 쓸 수 있는 건 무엇이고, 예측 시점에 "실제로 알 수 있는" 정보인가?

## 스스로 점검

각 문제의 (1) 학습 유형, (2) 회귀/분류 여부, (3) 타깃을 적어보세요.

1. 환자의 검사 수치로 당뇨병 여부를 판정한다.
2. 중고차의 연식·주행거리로 적정 시세를 매긴다.
3. 영화 리뷰 텍스트로 평점(1~10)을 예측한다.

3. 지도학습 / 회귀 또는 다중분류(둘 다 가능) / 타깃: 평점
