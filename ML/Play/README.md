## Google Colab으로 실행하기 (설치 0분)

이 노트북들은 **Google Colab**에서 바로 돌아가도록 만들었습니다.
파이썬, NumPy, Pandas, scikit-learn, Matplotlib이 **이미 설치돼 있어서**
내 컴퓨터에 아무것도 깔 필요가 없습니다. 구글 계정만 있으면 됩니다.

### 여는 방법 (가장 쉬운 순서)

1. 아래 표의 **"Colab에서 열기"** 배지를 클릭합니다.
   (GitHub에 푸시된 뒤부터 작동합니다.)
2. Colab이 열리면 상단 메뉴 **런타임 → 모두 실행** (또는 `Ctrl/Cmd + F9`).
3. 셀을 위에서부터 하나씩 `Shift + Enter`로 실행하며 출력을 관찰하세요.

> 💡 배지가 아직 안 열린다면(리포지토리가 비공개이거나 미푸시):
> [Colab](https://colab.research.google.com) 접속 → **업로드** 탭 →
> 이 폴더의 `.ipynb` 파일을 끌어다 놓으면 됩니다.
