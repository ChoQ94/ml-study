# 03. 머신러닝 기본 개념

> 로드맵 [3번 챕터](../../README.md#3-머신러닝-기본-개념)에 대한 강의입니다.
> 알고리즘에 들어가기 전, **반드시 잡아야 할 큰 그림**을 다룹니다. 이 챕터를 건너뛰면
> 나중에 "모델이 왜 안 되는지" 진단할 수 없게 됩니다.

## 이 챕터의 목표

이 챕터가 끝나면 다음 질문에 스스로 답할 수 있어야 합니다.

- 내 문제는 지도학습인가 비지도학습인가? 회귀인가 분류인가?
- "모델이 학습한다"는 말은 **수학적으로** 무슨 일이 일어나는 것인가?
- 훈련 데이터를 잘 맞히는 모델이 왜 실전에서 망가지는가?
- 데이터를 왜 굳이 훈련/검증/테스트로 나누는가?

## 학습 포인트 (이 챕터의 핵심 한 문장)

> **"훈련 데이터를 잘 맞히는 것"과 "처음 보는 데이터를 잘 맞히는 것"은 전혀 다른 문제다.**
> **머신러닝의 진짜 목표는 후자, 즉 _일반화(generalization)_ 다."**

이 문장이 왜 참인지를 몸으로 이해하는 것이 챕터 3 전체의 목적입니다.
